Par ce présent document, les informations liés à la génétique des clones vont être enfin rendus publique.
Cette génétique fut créer dès l'années 257 par le scientifique Soan NATANO pour empécher l'utilisation de civils dans les guerres.

## Découverte
Par décision des membres du haut conseil, voir [[Vote - Clones]], le chercheur est autoriser à prendre le sang de Rex GÜY, le clone original et possédant des capacités divines pour le transférer dans des nouveaux nés génétiquement modifier et gérer en laboratoire avec son équipe. L'expérience sera présenter par des témoins sur 2 expériences différentes.

### Test expérimental 1
> Le clone A possède une tête déformer, les conditions de vie dans la capsule sont défavorable. Seulement le clone A-4 à survécu mais l'intelligence n'est pas transférer correctement
>
> Le clone B ressemble parfaitement à Rex, jeune. Parmis les 5 témoins, seulement le clone B-6 peut vivre le plus longtemps avec 5 mois.
>
> Note: Trouver ce qui permet d'élargir la durée de la vie en respectant les décisions du haut conseil dans le test A, et trouver ce qui provoque la stabilité du corps dans le test B.
> Note: Les éléments présentes dans ces deux tests sont classifiées.

### Test expérimental 2
> Détails: Le clone A-4 est en ce moment mon assistant.
>
> Le clone A à parfaitement survécu comme d'habitude, mais avec les mêmes résultats, je pense que je me suis tromper de gênes depuis le test B.
>
> Le clone B est mort sur le coup lors de sa sortie de capsule, ces poumons n'était pas bien former, de même pour les autres témoin.
>
> Note: Réanalyser le code ADN modifié sans provoquer d'erreur.

### Test expérimental 3
> Le clone A à survécu, ainsi que ces témoins, leur durée de vie est normalement de 10 ans.
>
> Le clone B était dans de trop mauvaise conditions de vie, les clones n'ont pas survécu sauf le clone B-3.
>
> Note: Analyser l'ADN du clone B-3, améliorer le gêne des clones A.
