    Depuis github

    1. Création de votre fork

    Pour créer votre document, il vous suffit de vous rendre sur Github, via le lien https://github.com/Oria-Organization/oria-bot/ qui est notre repo. La création d'un compte est requis, voir (Documentation github).
    Par la suite vous pourrez cliquer sur la flèche de "fork" (voir image 1), et ensuite "Create a new fork" (voir image 2).


    Vous vous retrouverez sur la page de l'image 1, il vous suffit donc de faire "Create fork".

    Tout le contenu du repo sont sous la possession d'Oria Production, la modification du copyright où la publication du site est interdit.
    Vous êtes l'auteur du document, pour la contribution du site, tout retrait peut être discuter, les adminstrateurs auront toujours le dernier mot, voir (Droit d'auteur).

    Vous aurez enfin votre propre repo, ou plutot un fork (Vous pouvez le supprimez à tout moment via les settings puis général, section "Danger Zone").

    2. Comment créer votre document

    Pour créer un document il faut créer un commit, un commit est une sauvegarde des modifications de votre repo.
    Pour modifier le fork il y a plusieurs façon.

    1. Créer votre codespace.
    2. (Recommender) Créer un fichier séparemment et le télécharger sur le repo. Cliquez sur un fichier puis "ADD file".

    Le format du document sera expliquer dans le guide de rédaction.

    Depuis Discord

    Pour lancer une commande de notre bot, vous pouvez soit rejoindre notre discord, sois en installant le bot.

    Par la suite rendez vous dans un salon auquel vous avez les permissions de "Utiliser les commandes des applications" et "Envoyer des messages".
    Et faites comme les images ci dessous.


    Par la suite lancer la commande et remplissez les champs demander, pour le formulaire qui va s'ouvrir, pour le format et le remplir rendez vous sur le le guide de rédaction.
